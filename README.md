# http-helpers-ts

HTTP Helpers TypeScript library.

## Installation

```sh
npm install @evozong/http-helpers-ts
```

## Usage

```ts
import { hello } from "@evozong/http-helpers-ts";

console.log(hello); // "Hello World!"
```

## Development

Install dependencies:

```sh
npm install
```

Build:

```sh
npm run build
```

## Publishing

```sh
npm login
npm publish --access public
```
